# MDM Remote Control

This folder contains the remote-control feature design and shared protocol notes.

The feature is integrated into the existing projects only where required:

- Android APK integration lives in `hmdm-android` so the agent is packaged into one APK.
- Web/server UI integration lives in `hmdm-server` so the laptop dashboard can open a remote session.
- Feature ownership, protocol, and implementation notes stay here to keep the new work separate.
